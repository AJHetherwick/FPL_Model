# Fantasy Premier League Player Point Prediction

### Project Directory

#### Data

`DataScrape.ipynb`: Scrapes data from `FBRef.com`

`DataMerge.ipynb`: Merges FPL data with scraped data.

`DataAddTimeDecay.ipynb`: Prepares final features to input into the model.


#### Analysis + Modelling

`STA221_EDA.ipynb`: EDA of the dataset.

`Adams_Testing_Notebook.ipynb`: Lots of Analysis. Binary classifier model to predict if players will play.

`TotalPointsRegression.ipynb`: Point prediction regression model.
